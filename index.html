    Zadanie 4: Sposoby podłączania CSS do kodu strony WWW

    Sposób Opis
    Zewnętrzny arkusz stylów <link rel="stylesheet" type="text/css" href="style.css" />
    Styl lokalny <div style="color: red;">Tekst</div>
    Wewnętrzny arkusz stylów <style> h1 { color: blue; } </style>
    Alternatywny arkusz stylów <link rel="alternate stylesheet" type="text/css" href="style1.css" title="Nazwa 1" />
    Import arkusza stylów <style> @import url("style.css"); </style>
