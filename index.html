CSS - Cascading Style Sheets

CSS stands for Cascading Style Sheets. It is a language used to describe the style of document presentations in web development.

Nowe możliwości CSS

  1. Przejrzystość i krótkość dokumentów
  2. Łatwe zarządzanie serią dokumentów
  3. Oszczędność pisania dzięki klasom selektorów
  4. Różnorodne jednostki i sposoby definiowania kolorów

Wady CSS

  • Złożona specyfikacja
  • Brak sygnalizacji błędów
  • Błędne obsługiwanie przez różne przeglądarki

Zadanie 4: Sposoby podłączania CSS do kodu strony WWW

Sposób Opis
Zewnętrzny arkusz stylów <link rel="stylesheet" type="text/css" href="style.css" />
Styl lokalny <div style="color: red;">Tekst</div>
Wewnętrzny arkusz stylów <style> h1 { color: blue; } </style>
Alternatywny arkusz stylów <link rel="alternate stylesheet" type="text/css" href="style1.css" title="Nazwa 1" />
Import arkusza stylów <style> @import url("style.css"); </style>
